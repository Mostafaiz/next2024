# شروعی بر Next.js

خلاصه درس ها به صورت داکیومنت و `Code` جهت یادگیری و یادآوری نوشته می شود
ابتدا نود را نصب می کنیم.

دیدن نسخه نود در ترمینال 
```
node -v
 npm -v 
 yarn -v
 npx -v             //  npm install -g npm@10.4.0  
 npm list react      // یادم نیست:)

```

رفتن به پوشه قبل 
```
cd..
```


دستور نصب next
```
npx create-next-app@latest

```
